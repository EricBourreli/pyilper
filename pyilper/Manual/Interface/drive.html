Drive Tabs

 pyILPER Manual » User Interface

Description

This virtual emulates either a HP82161A cassette drive, a HP9114B disk drive or a virtual HDRIVE1 device to process files on a LIF image file.

The directory listing of the current LIF image file is shown on the tab. The listing is updated after write operations if the loop is idle for 3 seconds. The column headers can be checked to obtain a sorted list of the directory.

The virtual devices have the following HP-IL configuration:

HP82161A0x100x02

Use the HDRIVE1 drive type to work with LIF file images that can be downloaded from the internet. Use the HP9114B drive type if you use software that requires compatibility with that device. Use the HP82161A drive type if you like to use the virtual drive with the HP-41.

Note: PyILPER requires a valid lif header containig:

  • The LIF identifier 0x800 at offset 0
  • The LIF Version number 0x01 at offset 20
  • The media size information (number of tracks per surface, number of surfaces, number of blocks per track) starting at offset 24

If a LIF image medium is initialized pyILPER fixes missing header information.

Controls

The drive tab has the following controls:

  • LIF image file: Select disk file that contains the image of a LIF file. If a non existing file was specified the medium must be initialized by the controller. Note the file is only opened by pyILPER if read or write operations occur. The layout of the LIF image file must match the emulated drive. See the compatibility matrix above.
  • Drive active: If checked, the virtual HP-IL drive is available and can be used as HP-IL device. The HP-IL loop must always be reconfigured if this box was checked or unchecked.
  • Drive type: Choose the proper drive type that should be emulated (see above). Note that the LIF image file must be reopened, whe the drive type was changed.
