User Interface

 pyILPER Manual

General Overview

Before you start plug in the PIL-Box and establish the HP-IL loop with at least the controller (HP41 or HP-71B). After starting pyILPER you have to reconfigure the devices on the loop, by either power off and on the controller or by issuing a "RESTORE IO" on the HP-71B.

The user interface is composed of:

  • file menu to configure the program
  • tabbed virtual HP-IL devices on the main window
  • status prompt at the bottom of the main window

File Menu Entries

pyILPER configuration

PIL-Box: check this box if pyILPER is connected to a PIL-Box via a serial device. The serial device and the baudrate of the PIL-Box are configured here. The default for the device is /dev/ttyUSB0 on LINUX and COM1 on Windows. Note: the baud rate must match the hardware configuration of the PIL-Box. See the PIL-Box documentation for details.

TCP-IP (experimental): check this box to use the experimental feature of virtual HP-IL over TCP-IP. The configuration parameters are:

  • Port: the port to which the previous HP-IL device must connect
  • Remote Host, remote Port: the host and the port of the next virtual HP-IL device in the loop.

Configuration of the working directory: This is the location of the log files for the scope and the printer (q.v.). Default is the $HOME directory.

HP-IL device config

Configures the number of virtual HP-IL devices:

  • 0 to 5 virtual disk drives
  • 0 to 3 virtual printers
  • 0 to 1 virtual terminal

Note: the scope tab can not be removed from the application. If the number of devices was chaged you have to restart the applications to take the changes into effect.

HP-IL device status

Shows a pop up window with status information of the virtual HP-IL devices.

The status information includes the current HP-IL device id, the accessory id, the addesses (simple and extended) and the internal HP-IL status of the virtual device.

The status information is updated every 500 ms.

Reconnect

Restarts the application, if e.g. you forgot to plug in the PIL-Box on program start.

Quit

Terminates the application.

Virtual HP-IL device tabs

Each virtual HP-IL devices has its own tab:

  • Scope Tab
  • Drive Tabs
  • Printer Tab
  • Terminal Tab
