User Interface

 pyILPER Manual

General Overview

Before you launch pyILPER connect the PC to the PIL-Box and set up the HP-IL loop with at least the HP-IL controller (HP41 or HP-71B). After the launch of pyILPER you must reconfigure the HP-IL devices on the loop, by either powering the controller off and on again or by issuing a "RESTORE IO" on the HP-71B.

The user interface consists of:

  • The file menu to configure pyILPER
  • The utilities menu
  • The help menu
  • Tabbed virtual HP-IL devices: scope, printer(s), drive(s), terminal
  • Status information at the bottom of the main window

File Menu Entries

pyILPER configuration

PIL-Box checkbox

Check this box if pyILPER is connected to a PIL-Box via a serial device. The name and the baud rate of the serial device is configured here. See the setup section of the installation guide for details.

Note: the baud rate must match the hardware configuration of the PIL-Box. See the PIL-Box documentation for details. If the baud rate is set to "Auto" pyILPER tries to determine the baud rate automatically.

The "Enable IDY frames" option enables the PIL-Box to transmit the IDY frames to pyILPER. This causes an increase of traffic on the serial USB device. Use this option only if you use the virtual keyboard of the terminal emulator with the HP-71B.

TCP/IP checkbox

Check this box to use virtual HP-IL over TCP/IP. The configuration parameters are:

  • Port: the port to which the previous HP-IL device must connect
  • Remote Host, remote Port: the host and the port of the next virtual HP-IL device in the loop.

Configuration of the working directory

This is the location of the log files for scope and the printer output (q.v.). Default is the $HOME directory of the current user.

Configuration of the terminal, scope and printer display

You can set the size (80x24, 80x40, 132x24), the color scheme (white, amber, green), the maximum number of lines in the scroll up buffer and the font size. Display scrolling can be disabled if you set the number of lines to zero. You have to restart the application for take the changes to effect.

HP-IL device config

Configures the number of virtual HP-IL devices:

  • 0 to 5 virtual disk drives
  • 0 to 3 virtual printers
  • 0 to 1 virtual terminal

Note: the scope tab can not be removed from the application. If the number of devices was changed you have to restart the application for take the changes to effect.

Reconnect

Restarts the application, if e.g. you forgot to plug in the PIL-Box when launching the application.

Quit

Terminates the application.

Utilities Menu Entries

Initialize LIF image file

Initialize a LIF image file. Various media types, the number of directory entries an an optional label can be specified. This utility allows the initialization of media types which can not be created from the HP-71 by default.

Warning: do not initialize a LIF image file that is mounted to a virtual drive.

Note: this menu entry is only accessible if the latest version of the LIFUTILS are installed.

Fix header of LIF image file

Some HP-IL controller do not initialize the LIF image file header properly. pyILPER will issue a warning if the image file header does not contain valid medium size information. This menu entry fixes this information. Be careful to specify the proper medium type.

Warning: do not initialize a LIF image file that is mounted to a virtual drive.

Note: this menu entry is only accessible if the latest version of the LIFUTILS are installed.

HP-IL device status

Shows a pop up window with status information of the virtual HP-IL devices.

The status information includes the current HP-IL device id, the accessory id, the HP-IL address (simple and extended) and the internal HP-IL status of the virtual device.

The status information is updated every 500 ms.

Copy PILIMAGE.DAT to workdir

Copies the LIF image file that contains software to set up the terminal emulator for the HP-71B to the current working directory.

Check LIFUTILS installation

Display diagnostic information if a valid LIFUTILS version was found on the system.

Help Menu Entries

About

Manual

Shows the online manual in a browser window.
