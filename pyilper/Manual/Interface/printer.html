Printer Tab

 pyILPER Manual » User Interface

Description

This virtual HP-IL device emulates a dumb printer.

HP-IL configuration:

Accessory ID0x2E
Default Address3
Device IDPRINTER

The output of the printer is shown on the display and can be logged to a file.

The virtual printer does not support any escape sequences. Character mapping can be configured.

Device controls

The printer tab has the following controls:

  • Device enabled checkbox: If checked, the virtual HP-IL printer is available and can be used as HP-IL device. The HP-IL loop must always be reconfigured after activation or deactivation.
  • Log Printer checkbox: Activates writing printer output to the file "printer.txt" in the pyILPR working directory. Output is always appended to that file. A time stamp in the file marks the start and the end of each logging. The log file is closed automatically on program termination.
  • Character set combobox: select the character set that is emulated. For details of character mapping see the documentation of the terminal tab.
