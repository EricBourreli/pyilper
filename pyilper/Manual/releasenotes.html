Release Notes for pyILPER 1.5.0

  • The PILTERM utility was updated to version 2. You'll find the old version as PILTERM1 on the PILIMAGE.DAT image.
  • This version needs Qt5.
  • The font size of the drive directory listing can be configured. The row height of the listing is now properly adjusted to the font height.
  • See the Change Log for a full list of changes.
  • Improved on line help.
