Release Notes for pyILPER 1.7.0

  • The tabbed virtual HP-IL devices can be undocked and moved to floating, resizeable windows by pressing the undock button at the right corner of the tab bar. Clicking the close button of a floating window docks the virtual device into the tab again.
  • The configuration of a device tab/window is now implemented by cascading menus which can be called with the Configuration button in the bottom right corner. To clean up the user interface all configuration options with the exception of enable/disable device or logging on/off were moved to the cascading menu. Global parameters which affect all device windows or tabs in question are configured in the pyILPER configuration dialog as before.
  • The number of terminal columns and the color scheme are now configured individually in the configuration menu of the device tab/window.
  • A new configuration option allows to control whether log output is flushed to disk after an end of line was encountered. Turning buffering off will slow down the program considerably. This option can be configured individually for each device tab/window that supports logging.
  • Pixel size of HP82162A display is now configurable in the pyILPER main configuration dialog.
  • Terminal, printer, scope and HP82162A keep now their scroll position on resize.
  • The selection feature of the Terminal, Printer and Scope now supports autoscrolling.
  • Scope, printer and terminal tab/window got a display size indicator.
  • The file management functions of the drive tab/window are now accessible by push buttons. Note: These buttons are only accessilbe if the LIFUTILS are installed and the drive is disabled.

See the Change Log for a full list of changes.
