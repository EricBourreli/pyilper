Release Notes for pyILPER 1.6.0

  • pyILPER now automatically adjusts the number of lines of the printer, scope and drive directory output as well as the size of the plotter output to the size of the main window. The number of columns of the terminal, scope and printer output remains fixed and can be configured to either 80 or 120 characters (see below).li>
  • This implied the removal of the terminal size configuration parameter and the introduction of the new parameter terminal width.
  • The way to disable scrolling changed as well. Set the scroll up buffer size to favored number of lines. Then resize the main window until all lines are displayed. The minimum number of lines of the scroll up buffer is 24.
  • In addition to the position the previous pyILPER main window size is restored on start of the program.
  • The HP82162A printer is now included as virtual HP-IL device. It is not included in the default tab configuration. Thus, you must add it with the Virtual HP-IL device configuration in the file menu.
  • Barcodes can be created from the context menu in the drive tab for HP-41 program and SDATA files.
  • A PDF file of the directory listing can be created from the tool menu of the drive tab.
  • The paper size option was moved from the plotter configuration to the pyILPER configuration in the file menu. This parameter now controls the medium size of the PDF output as well as the plotter settings. Since this configuration parameter is global now you must reconfigure it if you used it in previous pyILPER versions.
  • See the Change Log for a full list of changes.
