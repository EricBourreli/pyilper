Release Notes for pyILPER 1.4.0

  • This version of pyILPER requires an updated version of the LIFUTILS. At least version 1.7.6 is required. If older versions of the LIFUTILS are installed the file management functions of the drive tab and the new virtual HP7470A plotter are disabled. See the download page to obtain the latest version for your platform. Note: on Windows (64bit) please deinstall a previous version of the LIFUTILS manually before running the installer.
  • The Virtual HP-IL device configuration was completely rewritten. No you can add or remove virtual HP-IL devices and change their tab position. If a new device is added you must assign a unique name. With the exception of the scope, the number of virtual devices is not limited. Note: settings of the virtual devices of previous pyILPER versions are not migrated completely to version 1.4.0. Please check them carefully.
  • pyILPER starts with a default configuration of a scope, a printer, two drives and a terminal. The printer as the first virtual device avoids avoid MANIO issues with the HP-41C. You may change that in the Virtual HP-IL device configuration.
  • New virtual HP-IL device: the HP7470A plotter.
  • See the Change Log for a full list of changes.
