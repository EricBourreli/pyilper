Welcome to pyILPER

This is the Readme for Version 1.1.0 of pyILPER.

Requirements

In order to run pyILPER, you must have the following software installed:

  • python >= 3.4 www.python.org
  • Qt >= 4.4 www.trolltech.com/download/index.html
  • PyQt4 www.riverbankcomputing.co.uk/pyqt/download.php
  • pyserial >= 2.6 http://sourceforge.net/projects/pyserial
The program won't work if the Qt libraries were built without threading support enabled.

Installation

There exists a Debian package to install pyILPER. On non Debian systems try to install the package with the command:

   python3 install.py install

as root. The installation procedure will install a package pyilper in the standard python package location and a script pyilper that starts the application.

The Debian package also provides an entry in the start menu.
